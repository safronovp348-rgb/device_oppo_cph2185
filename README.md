## ⚙️ PROJECT DESCRIPTION

This is a custom Android device project for OPPO A15 (CPH2185).

Includes:

- Device Tree (TWRP / AOSP base)
- Prebuilt Kernel (stock extracted)
- Recovery configs
- Experimental support
